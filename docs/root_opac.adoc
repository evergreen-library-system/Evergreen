Introduction
============

include::shared/about_this_documentation.adoc[]

This guide to Evergreen is intended to meet the needs of anybody
who uses Evergreen's Public Catalog, particularly library
workers in public services roles.
It is organized into Parts, Chapters, and Sections addressing key
aspects of the software. 

Copies of this guide can be accessed in PDF and HTML formats from
http://docs.evergreen-ils.org/.
